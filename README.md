Gomoku/Five In A Row
=============

     这个工程是一个五子棋小游戏，不考虑禁手等较高级的规则，
	 可以用来和同一局域网中的电脑进行联机游戏。
	 
	 每一局由两个player组成，同一局域网中支持多局游戏,
	 游戏中显示了每一步的具体情况，可以和对手聊天，
	 也可以放弃认输，结束游戏。
